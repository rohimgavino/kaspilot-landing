Cara Kerja

Setup Instan Kurang Dari 3 Menit

Tidak butuh hosting, tidak butuh pengetahuan coding. Anda hanya menyalin dokumen yang kami berikan.

1

Salin Template Spreadsheet

Buka link Google Spreadsheet master yang kami kirimkan, klik **File > Make a copy (Buat salinan)** untuk menyalin template bersih ke Google Drive pribadi Anda.

2

Deploy Web App

Buka menu Apps Script yang tersemat otomatis pada spreadsheet baru Anda, klik tombol **Deploy > New deployment** sebagai Web App hanya dalam hitungan detik.

3

Setup via Wizard & Mulai Catat!

Buka halaman setup interaktif untuk memasukkan PIN dashboard dan token Telegram. Hubungkan bot Anda dan mulailah mengirimkan catatan pertama Anda!
