Cara Kerja

Aktivasi Berlisensi, Data Tetap di Akun Anda

Setelah pembayaran terkonfirmasi, admin mengirim template, License Key, dan panduan. Pembeli cukup menyalin spreadsheet, memasang library resmi, lalu mengaktifkan bot dari Setup Wizard.

1

Bayar & Terima Paket Aktivasi

Admin mengirim link Make a Copy template, License Key pribadi, Script ID Library resmi, dan panduan singkat setelah pembayaran terkonfirmasi.

2

Copy Template & Pasang Library

Buat salinan spreadsheet ke Google Drive Anda, buka Apps Script, lalu tambahkan Library dengan identifier KasPilotLibrary sesuai panduan admin.

3

Deploy Web App

Deploy Apps Script sebagai Web App dari akun Google Anda sendiri, salin URL Web App, lalu buka halaman aktivasi KasPilot.

4

Isi Setup Wizard & Set Webhook

Masukkan License Key, token bot Telegram, dan PIN dashboard. Setelah tersimpan, klik Set Webhook dan KasPilot siap menerima catatan transaksi.

Akses Template Setelah Bayar

Link Make a Copy dan License Key tidak ditampilkan publik. Kirim bukti pembayaran, lalu admin akan mengirim paket aktivasi pembeli.

Butuh Bantuan Admin?

Hubungi admin jika perlu bantuan memasang Library, deploy Web App, mengisi License Key, token Telegram, PIN dashboard, atau Set Webhook.

Pertanyaan yang Sering Diajukan

Masih ragu? Berikut adalah beberapa jawaban atas pertanyaan yang paling sering diajukan calon pengguna.

Apakah data keuangan saya aman? ▼
Ya, 100% aman. Data keuangan Anda tersimpan langsung di akun Google Drive pribadi Anda sendiri (melalui Google Spreadsheet). Kami selaku penjual sama sekali tidak mempunyai akses ke data ataupun riwayat transaksi Anda.
Apakah saya perlu membayar biaya bulanan? ▼
Tidak. Template ini bersistem sekali bayar saja (one-time payment). Tidak ada biaya bulanan. Anda meng-host skripnya sendiri di layanan gratis Google Apps Script milik Anda seumur hidup.
Bagaimana cara kerja fitur Scan Struk & Suara dengan AI? ▼
Bot ini terintegrasi dengan API Key Gemini (dari Google) atau Groq. Anda hanya perlu membuat API Key gratis pada platform penyedia AI tersebut, lalu menempelkannya di Setup Wizard. Panduan lengkap cara mendapatkan API Key gratis sudah kami sediakan di dalam paket pembelian.
Apakah orang awam bisa memasangnya sendiri? ▼
Sangat bisa. Kami telah mendesain Setup Wizard interaktif yang membimbing Anda mengisi parameter. Anda hanya perlu mengikuti petunjuk teks dan video panduan kami yang berdurasi kurang dari 3 menit.
Bagaimana sistem License Key bekerja? ▼
Setiap pembeli mendapatkan License Key pribadi. Saat Setup Wizard dijalankan pertama kali, lisensi akan terikat ke spreadsheet pembeli sehingga akses tetap rapi dan tidak bisa dipakai sembarang salinan.
